# Portfolio - Ashier Lloyd Villaruel

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Dark Theme** - Modern dark UI with smooth transitions
- **Smooth Scroll** - Seamless navigation between sections
- **Mobile Menu** - Hamburger menu for smaller screens
- **Contact Form** - Get in touch section with email and phone

## Sections

1. **Header** - Introduction with name and title
2. **About Me** - Personal information and skills
3. **Services** - Web Design, UI/UX Design, App Design
4. **Projects** - Portfolio of work
5. **Contact** - Contact information and form

## Files

- `index.html` - Main HTML file
- `styles.css` - CSS styling
- `index.js` - JavaScript for interactivity
- Images - Project screenshots and profile photo

## Deployment

Deployed on Vercel - https://github.com/kraulagdkamotanan/portfolio

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
